
Q-7  How to install React Js on Windows, Linux Operating System? How to Install
NPM and How to check version of NPM?

Ans:- 
     --> Installing React.js on Windows:

     1. Install Node.js and NPM:
     --> Download the Node.js installer from the official Node.js website.
     --> Run the installer and follow the prompts. By default, this will also install NPM.

     2. Verify Installation:
     --> Open the Command Prompt and check the versions of Node.js and NPM:
     // check node version
     node -v 
     // check npm version
     npm -v 

     3. Install Create React App using npm:
     --> npx create-react-app app-name
     --> cd app-name
     --> npm start

     --> Installing React.js on Linux

     1. Install Node.js and NPM: 
     Open a terminal and use the following commands to install Node.js and NPM. The installation method depends on your Linux distribution.

     --> For Ubuntu/Debian:
     sudo apt update
     sudo apt install nodejs npm

     --> For CentOS/RHEL:
     sudo yum install -y epel-release
     sudo yum install -y nodejs npm

     2. Verify Installation:
     --> Check the versions of Node.js and NPM:
     node -v
     npm -v

     3. Install Create React App using npm:
     --> npx create-react-app app-name
     --> cd app-name
     --> npm start
